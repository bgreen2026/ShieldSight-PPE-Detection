# Dataset Information

## Dataset Name
Construction Site Safety Image Dataset (Roboflow)

## Source
Kaggle

https://www.kaggle.com/datasets/snehilsanyal/construction-site-safety-image-dataset-roboflow

## Original Dataset Classes
- Person
- Hardhat
- NO-Hardhat
- Safety Vest
- NO-Safety Vest
- Mask
- NO-Mask
- Safety Cone
- Machinery
- Vehicle

## Classes Used for This Project

This project will use only the following classes:

- Person
- Hardhat
- NO-Hardhat
- Safety Vest
- NO-Safety Vest

The remaining classes will not be used because they are outside the scope of this project. Limiting the number of classes allows the model to focus on detecting PPE compliance more effectively while keeping the project achievable within the course timeline.
